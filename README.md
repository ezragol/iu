### Installation
```
mkdir build
cd build

../configure
make
sudo make install
```

### Development
```
git clone https://github.com/onion-boy/untitled
cd untitled
```
run (with valgrind) (you have to install valgrind)
```
./rebuild valgrind
```
run (without valgrind) (you do not need to have valgrind installed)
```
./rebuild
```
